# Project-OOP-

Đây là bài tập nhóm môn lập trình hướng đối tượng

Quy tắc làm việc nhóm:
1) Hoàn thành đúng tiến độ, tôn trọng lẫn nhau
2) Các thành viên biết sử dụng github, khi up code thì đẩy vào nhánh(branch) của mình, sau đó nhóm trưởng duyệt và đẩy vào main
3) Đặt tên theo quy ước
    - Có ý nghĩa
    - Package: tất cả chữ thường
    - Class: Viết hoa chữ đầu tiên trong từ
    - Phương thức/ Thuộc tính: bắt đầu chữ thường, viết hoa chữ cái đầu tiên các từ tiếp theo
    - Hằng: Viết in hoa hết
4) Viết code chỉnh chu, có comment đầy đủ
